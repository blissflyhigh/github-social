## Badge Selection Rationale

| Badge | Reason |
|-------|--------|
| Build Status | Shows CI health, critical for CLI tools |
| Version | crates.io version for Rust package |
| License | MIT detected from LICENSE file |
| Language | Rust as primary language |
| Stars | Social proof for open source |
| Downloads | Usage metric from crates.io |
| PRs Welcome | Encourages contributions |
