- **Type**: Library
- **Language**: TypeScript
- **Registry**: npm
- **CI**: GitHub Actions

## Badge Selection Rationale

| Badge | Reason |
|-------|--------|
| npm Version | Primary distribution channel |
| Build Status | CI health indicator |
| Coverage | Code quality signal for libraries |
| License | MIT from LICENSE file |
| TypeScript | Shows type safety |
| Downloads | Monthly npm downloads |
| Stars | Community interest |
| PRs Welcome | Open to contributions |
