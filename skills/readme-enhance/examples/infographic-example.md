## Alternative: Feature-Focused Variant

For projects where architecture is less important:

```
Create a NotebookLM-style infographic for "github-social",
feature-showcase layout (1280x640),

**Header (top 20%):**
"github-social" bold title centered,
"Claude Code Plugin for GitHub Optimization" subtitle,
Version badge and stars count

**Main Features (center 60%):**
3-column showcase:

Column 1: Social Preview
- Large image generation icon
- "AI-Powered Images"
- "Capture project spirit"
- DALL-E, SD support badges

Column 2: Repository Metadata
- Document/tag icon
- "SEO Optimization"
- "Better discoverability"
- Description + topics

Column 3: README Enhancement
- Badge/chart icon
- "Marketing Badges"
- "Visual infographics"
- shields.io integration

**Call-to-Action (bottom 20%):**
Install command in code block style,
"Works with any GitHub repository" tagline,
GitHub and docs link icons

**Style:**
Gradient purple background, cyan accents,
Modern card-based design, subtle shadows,
Professional developer tool aesthetic
```
